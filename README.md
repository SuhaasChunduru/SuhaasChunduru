## PROJECTS

### Automated Brain Tumor Segmentation — BC-TSEA-UNet
> Research-grade medical imaging model, reliable enough to assist radiologists in tumor boundary review

`Python` `TensorFlow` `OpenCV` `U-Net` `Deep Learning`

- Engineered a Bias-Corrected Twin SE-UNet on the multi-modal BraTS MRI dataset (T1, T1ce, T2, FLAIR)
- Achieved Dice scores of **0.9585 (ET) · 0.9598 (TC) · 0.9302 (WT)**, with 0.92 IoU and 0.9999 specificity
- Designed a hybrid Dice + boundary loss with Sobel edge-based refinement
- Built a preprocessing pipeline: Z-score normalization, N4 bias field correction, histogram matching
